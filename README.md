# Stagecard Social Card Builder

A WordPress add-on plugin for creating branded social cards. Public visitors can upload a photo, move/resize it inside the circular photo area, and download a finished PNG.

Created by Olivia Kohring.

## Shortcode

```
[stagecard_social_card_builder]
```

The older shortcode also works:

```
[dhkc_social_card_builder]
```

## Admin

When Stagecard is active, this plugin appears under `Programs > Social Card Builder`. If Stagecard is not active, it appears as its own `Social Card Builder` admin menu.

## Updates

The plugin checks this GitHub repository's latest release. Attach a built ZIP named like `stagecard-social-card-builder-v0-4-1.zip` to each release.
